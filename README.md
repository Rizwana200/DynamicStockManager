# Dynamic Stock Manager

Dynamic Stock Manager is a Python program to help you manage your inventory easily. You can add new items, remove items, check stock levels, see which items are high-demand, monitor expiry dates, and get summaries by category. All data is saved in a CSV file, so nothing is lost.

## Features
1. Add Item – Add new items with name, quantity, category, expiry date, and popularity.
2. Remove Item – Remove items that are sold or no longer available.
3. Check Stock – View all items in your inventory along with details.
4. Restock Suggestion – Shows items that need to be restocked.
5. Expiry Alert – Alerts for items near their expiry date.
6. High-Demand Items – Lists items that are most popular or in demand.
7. Category Summary – Gives a summary of items by category.
8. Undo Last Action – Lets you undo your last add or remove operation.

## How to Run
1. Make sure Python 3 is installed on your system.
2. Keep `inventory.py` and `items.csv` in the same folder.
3. Open VS Code and open the folder `DynamicStockManager`.
4. Open a terminal in VS Code (Terminal → New Terminal).
5. Go to the folder in the terminal if needed:
   cd /c/Users/rizwa/OneDrive/DynamicStoreManager
6. Run the program:
   python inventory.py
7. Follow the menu to add, remove, or check items.
